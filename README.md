For this assignment, I chose the Alert Triage Workflow because it is an important part of a cloud security platform. Security teams often receive many alerts every day, and it can be difficult to identify which ones need immediate attention. My aim was to design a workflow that makes this process simpler and faster.

The main user is a cloud security engineer or analyst who works with platforms like Amazon Web Services, Microsoft Azure, and Google Cloud Platform. I designed three screens: the Alert Dashboard for viewing alerts, the Alert Details page for understanding the issue, and the Investigation screen for taking action.

My main focus was to reduce alert overload, improve clarity, and speed up resolution. The workflow can be measured by faster response times and fewer manual efforts. Overall, I wanted to create a simple, practical, and user-friendly design for security teams.
